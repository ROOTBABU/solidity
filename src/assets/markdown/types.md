# Types

<p>The Solidity programming language is statically typed, which means each variable must be
								defined by its type. Data types in Solidity can generally be categorized into three
								groups: <b>value types</b> , <b>reference types</b> and <b>mapping types</b>. We will
								examine the various
								<b>value types</b> in this article.
							</p>

## Value types

<p>Value types are types that store data directly in memory.</p>
<p>These types of variables are always <a href="https://www.javascripttutorial.net/javascript-pass-by-value/">passed by value</a>, which is why they are called value types since they are always copied when assigned to another variable or passed into a function.</p>
<p>value types in solidity are listed below:</p>
<code>boolean</code>
<code>integers</code>
<code>fixed point numbers</code>
<code>addresses</code>
<code>fixed-size byte arrays</code>
<code>rational and integer literals</code><br><br>
<code>enums</code>
